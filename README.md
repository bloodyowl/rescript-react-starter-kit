# ReScript React Starter Kit

> An opinionated starter kit for ReScript React

## Commands

### yarn start

Starts ReScript compiler in watch mode

### yarn server

Starts the development server

### yarn build

Builds and bundles the project

### yarn test

Runs the test suite
